CMD

  1. cd: change directory
  2. mdir: make directory
  3. dir: show directory
  4. git status: view status of files. added or unadded
  5. git init: create repository
  6. git add (name of file): track/add files
  7. git commit -m "name": commit changes
  8. git push: push file to github
  9. git clone "link": clone code
  10. git branch: create new branch
  11. git push -u origin main: push to main
  12. git log: commit and push history
